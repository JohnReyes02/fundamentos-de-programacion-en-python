# Fundamentos de Programación en Python

Este directorio contiene materiales introductorios para aprender a programar en Python, enfocado en desarrolladores.

## Contenido

*   **01 Introducción a Python para Desarrolladores**: Conceptos iniciales y configuración del entorno.

## Configuración del Entorno

1.  **Crear el entorno virtual**:
    ```bash
    python3 -m venv venv
    ```
2.  **Activar el entorno**:
    *   macOS/Linux: `source venv/bin/activate`
    *   Windows: `venv\Scripts\activate`
3.  **Instalar dependencias** (si aplica):
    ```bash
    pip install -r requirements.txt
    ```

## Estructura
El proyecto incluye un entorno virtual (`venv`) y materiales de código base.

## Tecnologías
*   Python 3
